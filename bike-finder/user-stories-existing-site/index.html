User Stories for an Existing Site - Vineet

  • As Vineet, I want to be able to see the weight of the bikes so that I can ensure I carry it up two flights of stairs.
  • As Vineet, I want a fast bike that will allow me to get to work quickly.
  • As Vineet, I want to see how many gears the bikes have so that I know it will accomodate the hilly terrain of Portland.
  • As Vineet, I want a sleek looking bike that will match my age and job status.
  • As Vineet, I want to be able to zoom in on the pictures of the bikes so that I can see the smaller details.
  • As Vineet, I want a bike with a basket so that I have a place for my briefcase.
  • As Vineet, I want to see the sizes of the bikes so that I can ride comfortably.
  • As Vineet, I want to be able to see reviews of the bikes so that I can feel assured that I'm making the right decision.
  • As Vineet, I want know if a bike is durable so that will be able to take potholes and other bumps in Portland.
  • As Vineet, I want a bike with nice handlebars that allow for a comfortable upright position.
  • As Vineet, I want a reasonably priced well-made bike so that I don't spend too much money on it.
  • As Vineet, I want a bike with warranty so that I don't have to worry about any accidents that may occur.
